# Mini Task Tracker

A mini task tracking application built to practice.

# Features

- Create new tasks
- Update tasks
- Delete tasks
- Simple user interface

# Clone(Download) in terminal

git clone https://github.com/Lifelight270/mini-task-tracker.git

# Navigate

cd mini-task-tracker

# Install dependencies:

npm install

# Run project

npm run dev

# Open broser or just follow the link

http://localhost:3000
